## Технологический стек

| Категория | Технология | Обоснование |
| :--- | :--- | :--- |
| **Backend** | Python / Go | Разработка серверной логики и API |
| **ML/AI** | TensorFlow / PyTorch | Разработка моделей детекции и анализа паттернов |
| **Облако** | Yandex Cloud | Локализация данных, низкая задержка, ML-сервисы |
| **СУБД** | PostgreSQL | Транзакционные данные (OLTP), консистентность |
| **Аналитика** | ClickHouse | Хранение и агрегация логов звонков (OLAP) |
| **Кэширование** | Redis | Миллисекундная проверка белого списка |
| **API** | gRPC | Двунаправленный стриминг аудиопотока |
| **Мониторинг** | Grafana / Prometheus | Отслеживание здоровья сервиса в реальном времени |
| **Контейнеры** | Kubernetes / Docker | Оркестрация и масштабируемость |
