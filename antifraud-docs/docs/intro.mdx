#  Anti-Fraud API Platform

## Обзор продукта
**Anti-Fraud Platform** — это антифрод-приложение для защиты пользователей от телефонного мошенничества (вишинга) в режиме реального времени. 
Система анализирует входящие аудиопотоки, выявляет признаки социальной инженерии и оповещает пользователя о мошенническом звонке, 
предотвращает кражу персональных данных либо денежных средств.

## Основные возможности
*   **Распознавание фрод-звонка в реальном времени:** Анализ голоса и поведенческих паттернов собеседника непосредственно в процессе звонка.
*   **Мгновенное информирование пользователя** об угрозе через Push-уведомления и звуковые сигналы.
*   **Группы безопасности:** Единый контур защиты для семьи или сотрудников организации с системой группового реагирования.
*   **Белый список:** Механизм исключения доверенных номеров из процедуры анализа для предотвращения ложных срабатываний.
*   **Управление инцидентами:** Ведение журнала вызовов, маркировка номеров и предоставление обратной связи для дообучения моделей.

## Ключевые бизнес-цели
1.  **Снижение ущерба:** Сокращение числа успешных случаев вишинга на 20% в течение первого года эксплуатации.
2.  **Защита активов:** Предотвращение выполнения пользователями навязанных мошенниками действий (снижение конверсии в мошенническое действие с 14.5% до 5%).
3.  **Лояльность и безопасность:** Повышение доверия пользователей за счет надежной защиты без глубокого погружения в технические детали.

## Целевая аудитория
*   **Конечные пользователи:** Городские жители (25–44 года), наиболее подверженные риску социальной инженерии.
*   **Корпоративный сектор:** Компании, обеспечивающие защиту сотрудников от целевого фишинга.
*   **Партнеры (B2B2C):** Банки и финтех-организации, использующие API системы для защиты транзакций своих клиентов.

## Архитектурный подход
Проект базируется на принципах микросервисной архитектуры с использованием высоконагруженных технологий передачи данных (gRPC) для стриминга аудио и современного стека машинного обучения для анализа угроз. Платформа спроектирована с учетом строгих требований к конфиденциальности (GDPR, 152-ФЗ) и обеспечивает высокую масштабируемость.
